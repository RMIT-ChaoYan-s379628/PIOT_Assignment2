Smart Library IoT application¶

Our group was appointed to the local council library to complete this automated library management system, which has the basic functions of borrowing books, returning books, and the accompanying image and voice recognition functions to view system statistics.

Below is a breakdown and documentation of our group for the entire project.

  • Console menu- based systems on RP & MP
    • Register and Login
    • Sockets sent from RP
    • Search a book:
    • Borrow a book:
    • Return a book:
  • Web dashboard
    • Admin features and RESTful API
    • Generate CLOUD data visualisation report
    • visual representation of the all book lending and return statistics
  • Facial Recognition
    • OpenCV Based recognition
    • Images Stored function
  • Challenging part
    • Voice/search feature
    • Object-detection feature
    • Unit test suite
